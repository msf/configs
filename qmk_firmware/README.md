I used qmk firmware to tweak the keymap of the ID80 keyboard.
using:
- https://beta.docs.qmk.fm/tutorial/newbs_flashing
- https://config.qmk.fm/#/id80/LAYOUT
- and `bin/qmk json2c id80_layout_msf.json`
